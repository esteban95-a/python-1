# Sistema de Gestión Académica - Momento 1

Este proyecto es una aplicación de consola desarrollada en Python que permite gestionar estudiantes, registrar sus notas y calcular promedios. Fue diseñado siguiendo principios de programación modular y control de versiones con Git.

## 🚀 Requerimientos Funcionales
- **Registro de Estudiantes:** Permite ingresar nuevos alumnos al sistema.
- **Gestión de Notas:** Registro de múltiples notas por estudiante usando diccionarios.
- **Cálculo de Promedio:** Función para visualizar el promedio y el estado (Aprobado/Reprobado).
- **Interfaz Interactiva:** Menú basado en un ciclo `while` para navegación continua.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** Python 3.x
- **Control de Versiones:** Git y GitHub
- **Estructuras de Datos:** Listas y Diccionarios

## 📂 Estructura del Proyecto
- `app.py`: Punto de entrada de la aplicación e interfaz de usuario.
- `funcionaprueba.py`,`datos.py`,`funcionnotas.py`,`funcionregistro.py`: Contiene las funciones de procesamiento de datos y lógica de negocio.
